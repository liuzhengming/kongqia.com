> var arr = [1,2,3];

1. arr instanceof Array

2. arr.constructor === “Array”

3. Object.prototype.toString.call(arr) === “[object Array]”

4. Array.isArray(arr)

完美写法如下：

```
function isArr(arr){
  if(typeof Array.isArray === "Function"){
    return Array.isArray(arr)
  }else{
    Object.prototype.toString.call(arr) === "[object Array]"
  }
}
```
